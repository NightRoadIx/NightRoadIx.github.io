EJEMPLO ESTRUCTURA BÁSICA HTML

Esqueleto básico

<!-- Se le indica que el tipo de documento es html -->

<!DOCTYPE HTML>

<!-- Se colocan las etiquetas para decir que todo lo que sigue es código html -->

<html>

 <!-- Aquí colocamos la cabeza del documento HTML -->

 <head>

  <!-- Un título para nuestra página -->

  <title> Título de la página Web </title>

  <!-- Información para el navegador -->

  <meta name="description" value="Descripción de mi página para mostrar en el buscador" />

  <meta name="keywords" value="Palabras clave con las que los usuarios llegarán a mi página web a través de los buscadores" />

  <meta name="author" value="Autor de la página web" />

 </head>

 <!-- Aquí colocamos el cuerpo del documento HTML -->

 <body bgcolor="color de fondo" text="Color del texto" >

  <!-- Para ir creando párrafos dentro del documento -->

  <p> Párrafo dentro del cuerpo del documento html </p>

  <p align="left"> Párrafo alineado a la izquierda </p>

  <p align="center"> Párrafo alineado al centro </p>

  <p align="right"> Párrafo alineado a la derecha </p>

  <p>Letra en <i>itálica</i>, <b>negritas</b>, <u>subrayada</u>

  <p> Otras modificaciones en texto <big>más grande</big>, <small>más pequeño</small>, <mark>marcado</mark> </p>

  <p> Seis diferentes tamaños de títulos: </p>

  <h1> Tamaño muy grande </h1>

  <h2> Tamaño grande </h2>

  <h3> Tamaño normal grande </h3>

  <h4> Tamaño normal pequeño </h4>

  <h5> Tamaño pequeño </h5>

  <h6> Tamaño muy pequeño </h6>

  <p> Línea de separación: </p>

  <hr align="alineación de la separación" size="Grosor de la línea" width="Ancho de la línea" color="Color de la línea"/>

 </body>

</html>
